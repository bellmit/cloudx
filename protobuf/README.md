#### Google Protocol Buffer协议封装类
- 支持对JAVA基本类型、对象、Map、List数据序列化&反序列化
- 可将序列化的数据进行网络传输、JVM内存存储、REDIS存储等，提供统一的数据操作接口
- 详细示例查看test测试案例
